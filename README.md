#*Телеграмм бот сделанный для обработки заказов на TELEBOT
Написанн на Python, сделанн для ROFSID от ROFSID
##Что он делает?
- 1. Позволяет собирать все обращения в одном боте, добавляя к каждому отправленному сообщению хештег (Позволяет удобно сортировать сообщения в самом Телеграмм)
- 2. Обрабатывать творческому автору заказы в зависимости от его типа, в данном боте это заказ на: арт, 3д модель, сайт, бота, другое
- 3. Оставлять свои личные контакты анонимными, что позволяет автору написать заказчику напрямую тогда, когда это будет удобным
